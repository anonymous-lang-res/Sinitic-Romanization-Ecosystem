> **Important:** The `MeixianKejiaRomZJ2_symbol.json` file should be imported into PhonSymbol using the **medial–nucleus–coda (M–N–C, 頭腹尾) mode**. Using the **Final mode** can result in incomplete import and display of the scheme. In future versions, we will add a warning when an incompatible mode is selected.

MeixianKejiaRomZJ1 is a romanization system for Meixian Hakka (Kejia) that we uploaded on July 1. Prior to the paper submission deadline of June 21, Anywhere on Earth (AoE), we had uploaded another romanization system for Meixian Hakka, MeixianKejiaRomZJ2.

Both MeixianKejiaRomZJ1 and MeixianKejiaRomZJ2 are designed for Meixian Hakka, but they adopt different romanization conventions and design approaches. As part of this long-term project, we will continue to develop, refine, and expand the romanization ecosystem, regardless of whether updates are made before or after the paper submission deadline.
